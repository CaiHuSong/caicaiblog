# caicaiblog
vue+springboot的前后端分离多人博客项目

### 使用的技术
* 后端
  * springboot
  * mybatis-puls
  * redis
  * mysql
  * jwt
  * 七牛云
* 前端
  * vue
  * elementui
  * axios
  * router
